## Proyecto
Juego de Asteroides 2D clásico en HTML5 Canvas puro (sin dependencias).
- Archivo principal: `game.js` (todo el juego en un solo archivo)
- Canvas fijo: 800×600 px, se abre directamente en el navegador
- Rama de power-ups: `01-Powerups`
- Power-ups implementados: Escudo (S, hexágono), Triple disparo (T, triángulo), Disparo rápido (R, diamante)
